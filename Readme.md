# 📊 Sales Performance Dashboard | Power BI

An interactive Power BI dashboard designed to analyze sales performance across regions, products, and customer segments.  
It provides both high-level KPIs and detailed insights to support business decision-making.

---

## 🎯 Objective

- Monitor key KPIs (Revenue, Orders, Profit, Return Rate)
- Analyze sales trends over time
- Evaluate product and category performance
- Understand customer behavior
- Compare actual performance with targets

---

## 🛠️ Tools & Technologies

- Power BI
- DAX (Data Analysis Expressions)
- Power Query
- Excel / CSV

---

## 🚀 Key Features

- Interactive slicers (Region, Year)
- Drillthrough from Top Products to Product Detail page
- Dynamic field parameters for flexible analysis
- Advanced DAX measures (time intelligence, targets, rolling metrics)
- Clean and intuitive dashboard design

---

## 📈 Key Insights

- Accessories category generates the highest number of orders
- Revenue shows consistent growth after 2021
- A small set of products contributes significantly to total profit
- Customer segments show distinct purchasing patterns
- Return rate is low (~2%), indicating good product satisfaction
